# The Atrium Below The World

`The Atrium Below The World` is a new original 1980s mallcore / underwater vaporwave track for the Ableton MCP project.

## Source Boundary

- No previous master is edited.
- No previous stem is reused.
- No ballroom/78rpm recordings are used.
- No prior vocal sources are used.
- No `Mall at the End of Sleep` source files are used.
- No user-provided source audio is read, copied, saved, or transformed.
- No external samples are downloaded or used.

The renderer is procedural-only and writes `sourceSamplesUsed: 0` in its verification report.

## Concept

The track imagines a dead 1980s mall whose glass atrium has become an underwater cavern. The emotional center is slowed vaporwave nostalgia: warm e-piano chords, submerged PA fragments, empty food-court air, and a mallet motif that feels like a half-remembered arcade or pool-level melody without quoting any game music.

The references are mood references only. The piece does not copy melodies, motifs, samples, sound effects, or recognizable material from any game or soundtrack.

## Sound

- Warm but haunted 1980s e-piano mall chords.
- Buoyant underwater mallet motif with slow current-like pitch drift.
- Submerged PA memory and VHS-style public-space haze.
- Alien atrium pads with suspended minor seconds.
- Cold choir and horn shadows with subtle Nordic color, not heroic scoring.
- Mono-centered deep pressure, sparse dead-escalator metal, and a controlled final bloom.

## Safety Boundary

The renderer is offline-only. It does not start Ableton, move the mouse, use LiveAPI writes, install plugins, download samples, scrape arbitrary URLs, rip YouTube/SoundCloud audio, expose network services, or create intelligible subliminal/coercive speech.
