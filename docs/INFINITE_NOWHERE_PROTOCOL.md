# Infinite Nowhere Protocol

`Infinite Nowhere Protocol` is an original offline-rendered horror/liminal track for the Ableton MCP project.

## Intent

The track is built around an impossible empty place: dead fields, service tunnels, fluorescent sky, no horizon, and a damaged memory trying to stay intact. The classified experiment / mind-control framing is fictional atmosphere only. The render does not include intelligible commands, coercive speech, or real subliminal instruction content.

## Sound Design

- Far 78rpm ballroom fragments act as the remembered melody.
- Public-domain female vocal fragments become a distant siren-memory hook, then collapse into vowel ghosts and reversed smears.
- Low-passed wind, concrete room tone, fluorescent hum, and lab-machine tones create the infinite nowhere environment.
- Sub pressure stays centered and controlled, with a late low-end absence followed by one final pressure swell.
- Tape artifacts are short field-recorder scars and dropouts, not a broadband static bed.
- Stereo motion is wide and unstable in reflections and vocals while the low end stays mono-compatible.

## Structure

1. False calm: beautiful but lonely memory in the distance.
2. Dislocation: pitch drift, widening room, unstable center.
3. Infinite nowhere: a short vocal-memory motif folds over itself.
4. Experiment: quiet clinical tones and fictional transmission chirps.
5. Bad-trip collapse: near-ear nonverbal vowels and smeared ballroom fragments close in.
6. Final void: low end drops out, then returns as one controlled sub-pressure swell.

## Render Command

```powershell
npm run render:infinite-nowhere-protocol
```

## Outputs

- `%USERPROFILE%\Downloads\infinite-nowhere-protocol-verification.json`
- `samples/staging/infinite-nowhere-protocol/stems/*.wav`

The staging folder is ignored by Git and is intended for local Ableton import/editing.

## Safety Boundary

The renderer is offline-only. It does not start Ableton, move the mouse, use LiveAPI writes, install plugins, download samples, scrape arbitrary URLs, or expose network services.
